About:

This website was mocked up with Figma then built using HTML5, CSS3, Javascript. I also implemented several Javascript libraries such as jQuery for state handling, GSAP for animations, and zdogJS for the pseudo-3D render. The asteroids were created with Vectary and managed in the canvas with ThreeJS. I utilized Git for version control and Trello for project management.


If you would like to see the source code please checkout my Github.


I would like to give a special thanks to Eddie Sun, Nirosh Manohara, Tony Ilersich, Sonya Yan, and Brandon Chu for motivation and inspiration.
