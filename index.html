About my website:

This website was built entirely from scratch by me using HTML5, CSS3, Javascript (that's right, no boostrap). I also implemented several Javascript libraries such as Jquery for state handling, GSAP for animations, and Zdog.JS for the pseudo 3D model. I utilized Git for version control and Trello for project management.


I plan to iterate on this project to continually make it unique and enjoyable for the eyes. If you would like to follow my progress, please take a look at my Github or Trello!
